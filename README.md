A secure, classified database system for the Council of Ricks. This Fullstack application allows agents to track entities across the multiverse, maintain dossiers, and monitor galaxy locations.

## ‚ö° Features

* **Multiverse Authentication:** Secure login system with bcrypt encryption.
* **Live Entity Feed:** Search and filter characters by status, gender, and name.
* **Voice Interface:** Text-to-speech integration for reading agent dossiers.
* **Galaxy Map:** Interactive visualization of character origins using calculated coordinates.
* **Classified Notes:** Agents can attach private, encrypted notes to any entity.
* **Agent Profile:** Personal dashboard with avatar upload (File System storage).
* **Admin Clearance:** Level 1 admins can monitor registered agents.
